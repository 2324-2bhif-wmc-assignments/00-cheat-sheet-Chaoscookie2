Html Tags

DOCTYPE

Is not a real tag, but is mandatory for every HTML file. Contains information about the file type.

html

The root tag for every HTML file. Is mandatory.

head

Must be included and often contains metadata and links to CSS files. Is always the first tag inside the html tag.

body

Contains everything that should be displayed on the website. Is always the second tag inside the html tag.

meta

Placed inside the head tag and is used to add metadata to the website. It doesn't display anything.

link

Placed inside the head tag and is used to link an external CSS file.

title

Is a mandatory tag, that is placed inside the head tag. Is displayed as the name of the website.

article

Can be used to group text as one article. The content should make sense on its own. The tag itself doesn't change the appearance of it's content on its own.

paragraph

Is used to group text as one paragraph. Most web browsers automatically insert a single blank line before and after the tag.

div

Is used to group text for formatting with CSS. Not visible on its own.

span

Like div, but is an inline element opposed to a block element.

a

Used to create hyperlinks, or as a literal anchor to which a hyperlink can lead. using the button tag and the onclick attribute, buttons can be used as links aswell

pre

All formatting applied to the text in this tag can also be seen on the webpage.

br

Breaks the line / starts a new line.

hr

Draws a horizontal line, used to separate independent text.

comments

Using "<!-- -->" the text inside can be commented To write the characters < and >, &lt; and &gt; have to be used

img

With the "src"(containing the actual image) and "alt"(containing text that replaces the image if it can't be loaded) attribute, images can be displayed

Lists

Unordered Lists

Start tag: <ul>, list item: <li>. Their content is listed without numbers - only symbols.

Ordered Lists

Start tag: <ol>, list item: <li>. Their content is listed with numbers.

Definition Lists

Start tag: <dl>, description: <dt> data: <dd>. Their content is listed with numbers.

table

There is the table head <thead> and <tbody> and <tfoot>. The body contains rows are made with <tr>, which contain table cells: <td>. The header contains <th> (header cells) the footer also contains <td> (table cells);
