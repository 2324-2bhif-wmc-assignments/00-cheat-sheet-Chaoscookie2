The World Wide Web

General

The world wide web was founded by Tim Berners-Lee, and was meant to be a decentralized file system with links between different databases on independent computers. The most important technologies to make this work are html, url and html.

HTML

HTML (Hypertext Markup Language) is used to form the base structure and content of a website.

URL

A URL (Uniform Resource Locator) is the address used by web clients to send HTTP requests to web servers. A URL's syntax typically looks like a URI (Uniform Resource Locator), which look like this:"[scheme]://[authority]/[path]?[query]#[fragment]".

HTTP

(Hypertext Transfer Protocol) is the primary protocol, by web servers and web clients to transfer hypertext. The secured version of it is called HTTPS.

Web Server

Web Servers are what make up the entire World Wide Web. They can receive HTTP request and can then send back either static content in the form of a preexisting file, or dynamic content which a web client can use to generate something like an HTML file on the fly.

Web Client

Web clients, also commonly known as browsers, are a way for people to access websites. When the user inputs a URL, the client sends an HTTP request to the specified server, and display the received resources.

Important HTML tags

DOCTYPE

Is not a real tag, but is mandatory for every HTML file. Contains information about the file type.

html

The root tag for every HTML file. Is mandatory.

head

Must include and often contains metadata and links to CSS files. Is always the first tag inside the html tag.

body

Contains everything that should be displayed on the website. Is always the second tag inside the html tag.

meta

Placed inside the head tag and is used to add metadata to the website. It doesn't display anything.

link

Placed inside the head tag and is used to link an external CSS file.

title

Is a mandatory tag, that is placed inside the head tag. Is displayed as the name of the website.

article

Can be used to group text as one article. The content should make sense on its own. The tag itself doesn't change the appearance of it's content on its own.

paragraph

Is used to group text as one paragraph. Most web browsers automatically insert a single blank line before and after the tag.

div

Is used to group text for formatting with CSS. Not visible on its own.

span

Like div, but is an inline element opposed to a block element.

a

Used to create hyperlinks, or as a literal anchor to which a hyperlink can lead.
