CSS

CSS Rule

A CSS rule consists of the selector like an element or a class and a declaration in curved brackets. Multiple selectors can be used by like this: h1, h2, p {declaration}

Comments

Comments start with /* and end with */

Colors

the color: property can be used to change the color of many things like text or background, using either predefined names or hex codes

Background

Using the background property you can implement background color and images with the background-repeat property you can make the image repeat only along one axis or not at all using the background-position property you can set a fixed position of an image You can also add a gradient with background-image: linear-gradient(); more on this later

Text

With the color property you can change the color of text. With the text-align property, the text can be aligned in the center, right or left. With text-align: justify; you can create block text. Using the text-decoration-line property you can add decorations like underlines. You can also color them using text-decoration-color. The text-transform property is used to specify uppercase and lowercase letters in a text.

Fonts

Serif fonts have small strokes at the end of each letter, Sans serif fonts don't. You can change the font with the font-family property. The font-style property is used to make text italic. The font-size attribute can change font size either in pixels or with em this takes the current font size of the browser as base, which by default is 16px. font-weight is used to make text bold.

Links

Links can be styled with the a:link, a:visited, a:hover and a:activate selectors.

Combinators

Using combinators you can select elements that have a certain relationship to other elements.
  • "space" : descendant selector, selects all elements of a type, an element of the first type
  • ">" : child selector, selects all direct descendants of a type that, that are within an element with the first type
  • "+" : adjacent sibling selector, selects elements of a type, that are directly under an element of the first type
  • "~" : general sibling selector, selects elements of a type, that are under or above an element of the first type example:
    p > span{
        background-color: yellow;
    }

Pseudo Classes

A pseudo-class is used to define a special state of an element. Written directly after the selector, common pseudo classes include: :link, :visited, :activated for the "a" tag, and :hover which also works for different ones. The :first-child pseudo-class selects an element, that is the first child of the selected tag for example p:first-child selects the first child in ever paragraph

Tables

You can change the border with the border: property. You can change things like width, color and visibility with it. border-radius: can round the corners. You can also merge the border's lines to remove the space between cells using border-collapse: collapse. Using the rowspan and colspan attributes, you can make one cell take up more space

Lists

Change ul bullet: list-style-type:; change bullet to image: list-style-image: url(""); change ol bullet: list-style-type:; Set bullet position list-style-position: inside/outside. For ol a bullet point may be upper-roman or lower-latin usw.

Box model

Every element has a display property. It can be block or inline or something in between. Block elements can have margin and padding in all directions, while inline elements only have them on the sides. Block normally create a line break

Flex

Flex first needs a flex box (for example div with display:flex).
properties for container
  1. flex-direction: can be column, row and both can be reversed
  2. flex-wrap: determines how the items wrap if not enough space
  3. flex-flow: combines the two above
  4. justify-content: changes how the elements are distributed along the main axis (for rows that is horizontal)
  5. align-items: changes how the elements are placed in line along the secondary axis
  6. align-content: changes how the elements are distributed along the secondary axis
  7. gap: changes exactly how much space is between the elements
Properties for children
  • order: normally each element is numbered like they are put in html. This number can be changed to change the order
  • flex-grow: Normally each element has the same amount of space. This number changes how many elements worth of space the element gets
  • align-self: used to override the align-items property for a single element

Float

Floating an element places it outside the normal text flow. For example with float: left, an element will float on the left side. If clear: right is used, text will NOT flow around it on the right side.

Gradients

For radial gradients, you can set how far it goes with closest/farthest-corner/side. You can also determine the center for it and the conic gradient with "at".

Positioning

  • static: normal text flow
  • relative: like static, but you can add an offset with top, right, bottom, left. This does NOT change the text flow for the surrounding elements
  • absolute: Positioned relative to it's closest positioned ancestor (with position static)
  • fixed: Has a fixed position in on the page, remains there when scrolling. Position can be changed with top, right, bottom and left properties
